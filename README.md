# Game of Life with Rust+WebAssembly

This is a toy project for learning how to use WASM, following the tutorial at https://rustwasm.github.io/docs/book/game-of-life/

## 🚴 Usage

### 🛠️ Build with `wasm-pack build`

```
wasm-pack build
```

### 🛠️ Run locally

```
cd www
npm install
npm run start
```
